# Summaries

## How to install ?

* **Clone the repo**
    ```bash
    git clone https://github.com/KonstantinZhukovskij/exchange-of-notes.git
    ```
* **Enter the folder**
     ```bash
     cd client
     ```
* **Install dependencies**
     ```bash
     npm install
     ```
* **Enter the folder**
     ```bash
     cd server
     ```
* **Install dependencies**
     ```bash
     npm install
     ```
* **Run project**
    ```bash
    npm start
    ```
* **Visit [localhost:3000](localhost:3000)**

## Used:

* **Node.js**
* **Express**
* **React**
* **PostgreSQL**
